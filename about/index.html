CoderDojoとは

CoderDojo は7〜17歳の子どもを対象にしたプログラミング道場です。2011年にアイルランドで始まり、世界では65カ国・870の道場、日本では全国に27の道場があります (2016年3月現在)。
CoderDojo枚方では、主に次のような内容をサポートする予定です。

Scratch

ビジュアルプログラミング言語による簡単なゲームの作成

HTML, CSS, JavaScript

Webサイトの制作やブラウザで動作するゲームの作成

Swift, Java

iOS, Androidを利用したスマートフォンアプリの作成
